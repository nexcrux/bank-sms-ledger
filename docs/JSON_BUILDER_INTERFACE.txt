Q: "I don't see variable options when I tap Value"
A: Make sure you selected "Dictionary" first. Variables only
   appear for Dictionary values.

Q: "Should I add quotes around the key names?"
A: No. Just type: body (not "body")
   Shortcuts adds quotes automatically in the final JSON.

Q: "The values show black text, not blue pills"
A: Delete the value and tap again. Make sure to SELECT from
   the popup menu, don't type the variable name.

Q: "Can I use Text instead of Dictionary?"
A: No. "Text" creates static text. "Dictionary" creates
   key-value pairs and allows variables.
